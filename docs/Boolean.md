### Boolean Operations
Bant also supports boolean types, ```true``` and ```false```, with the following operations:
- Less than ```<```
- Greater than ```>```
- Not ```!```
- Equals ```==```
- Not equal ```!=```
- Less than or equal to ```<=```
- Greater than or equal to ```>=```
- And ```&&```
- Or ```||```

Examples:

```
printBool(!true || !false && !(!true))
``` 
=> ```true```

<br>

```
printBool((5 > 4) != false || (3 + 4 < 5))
``` 
=> ```true```
